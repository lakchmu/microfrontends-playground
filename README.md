# 📦 Microfrontend React Application

## 🚀 Description

This fun generator of humorous excuses was created to explore the basics of microfrontend applications.

## 🗂️ Project structure

```
microfrontends-playground/
├── conainer/             # Container
├── excuse-generator/     # Generator of excuses
```

## ⚙️ Technologies

- React
- Webpack / Module Federation
- Tailwind CSS

## 🚩 Run project

### Local run container

```bash
cd conainer
npm run start
```

http://localhost:3000/

### Local run generator

```bash
cd excuse-generator
npm run start
```

http://localhost:3001/
